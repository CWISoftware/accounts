### Install environment

Install the [ASP.NET 5](https://github.com/aspnet/Home#cmd)

Open new shell and execute command about to install runtime:

	dnvm update

### Running first time

Refresh packages from nuget:

	dnu restore

### Run web application

Windows:

	dnx . web

Mac\Linux:

	dnx . kestrel
